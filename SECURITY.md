# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in Saykai, please report it responsibly.

**Do not** open a public GitHub issue.

Instead, contact us at:

**security@saykai.com**

We will acknowledge receipt within 2 business days and work with you to assess and resolve the issue.

## Supported Versions

Only the latest released versions of Saykai components are supported with security updates.

## Scope

This policy applies to:
- Saykai runner
- Saykai CI action
- Related tooling published under the saykai-systems organization
